# FitWear E-Ticaret Sitesi - Front-End

Gymshark, Jack Jones ve Under Armour tarzında premium fitness giyim e-ticaret sitesinin front-end temeli.

## Özellikler
- ✅ 3D efektli sabit navbar & footer
- ✅ responsive tasarım
- ✅ Sonsuz kaydırma (infinite scroll)
- ✅ Dinamik sepet sistemi (localStorage)
- ✅ Adet kontrollü ürün ekleme
- ✅ 3D hover efektli ürün kartları
- ✅ Çok dilli hazırlık
- ✅ Parallax hero section
